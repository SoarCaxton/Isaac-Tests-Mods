# 模组功能介绍

## RoomChances

用于测试特定楼层、特定房间的生成概率

## chests

用于测试生成普通箱子/金箱子时，发生掉落物替换的概率

## tr

用于测试标记石头/超级标记石头生成掉落物的概率

## BossDirection

用于统计各楼层：

1. 最终头目房在初始房间的方向
2. 各楼层初始房间，选择哪个门前进离最终头目房更近

## BossDirectionXL

用于统计(合层):

1. 各楼层首个头目房（虚空层选精神错乱房间）在初始房间的方向
2. 各楼层初始房间，选择哪个门前进离首个头目房（虚空层选精神错乱房间）更近

## BossDirectionKp

用于统计角色拥有刀柄时：

1. 4c/4d层最终头目房在初始房间的方向
2. 4c/4d层初始房间，选择哪个门前进离最终头目房更近

## BossDirectionXLKp

用于统计(合层)角色拥有刀柄时：

1. 3c/3d层首个头目房在初始房间的方向
2. 3c/3d层初始房间，选择哪个门前进离首个头目房更近

## BossDirectionItems

用于统计携带指定物品时：

1. 各楼层最终头目房在初始房间的方向
2. 各楼层初始房间，选择哪个门前进离最终头目房更近

## BossDirectionXLItems

用于统计(合层)携带指定物品时：

1. 各楼层首个头目房（虚空层选精神错乱房间）在初始房间的方向
2. 各楼层初始房间，选择哪个门前进离首个头目房（虚空层选精神错乱房间）更近

## BossDirectionItemsKp

用于统计携带指定物品和刀柄时：

1. 4c/4d层最终头目房在初始房间的方向
2. 4c/4d层初始房间，选择哪个门前进离最终头目房更近

## BossDirectionItemsXLKp

用于统计(合层)携带指定物品和刀柄时：

1. 3c/3d层首个头目房在初始房间的方向
2. 3c/3d层初始房间，选择哪个门前进离首个头目房更近

## BD.py

用于处理BossDirection系列模组的实验数据*.dat，生成包含实验数据、概率统计、归一化比例、门概率图和方向图的Excel文件。
